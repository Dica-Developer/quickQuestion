Quick Question

Communication and Collaboration Tool

Quick Question is an easy-to-use collaboration tool. It helps you to share your ideas with other Quick Question users simply by starting the application. You can easily ask or answer questions, collaboratively sketch on a whiteboard and share files.

Download

Our current release is version 0.10.13. This is the first pre-release. It is feature complete for version 1.0 and runs on Linux, Mac and Windows. At the moment, we are unable to provide a package for Windows, but we are working on resolving this before we launch version 1.0. However, you can easily build a Windows version (and all other versions) from the source. See the README.md file on github for build instructions.
