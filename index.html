Quick Question

Required Software

You need to have node.js installed, to start working with QuickQuestion.

Installation

After installing node.js, you have to run the following commands:

git clone git@github.com:Dica-Developer/quickQuestion.git
cd quickQuestion
npm install
npm -g install grunt-cli

Build

To build a distribution run the following commands corresponding to your operating system:

  1. For Mac OS:

    grunt dist

  2. For Linux:

    grunt dist-linux

  3. For Windows:

    grunt dist-win

The app can then be found under:

./dist/
